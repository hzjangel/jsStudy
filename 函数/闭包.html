# 闭包

# 定义

一个函数(fun1)里面还有函数(fun2),并且fun2使用了fun1里面的变量,称之为闭包

一个简单的例子如下

function fun1() {
    let a = 0;
    function fun2() {
        a++;
        console.log(a);
    }
}

闭包一般不会这样用,在使用闭包的时候,一般是将函数作为返回值(在JavaScript里面函数是Function的实例,即是一个对象,所以可以将函数直接作为返回值)

例子如下

function fun1() {
    let a = 0;
    return function() {
        a++;
        console.log(a);
    }
}

# 闭包的生命周期

一个函数在执行完成之后,函数里面的内容将会被销毁(作为垃圾进行回收),但使用闭包却不一样,会延长函数的生命周期

function fun1() {
    let a = 0;
    return function() {
        a++;
        console.log(a);
    }
}

let fun = fun1();
fun();  //1
fun();  //2

在使用闭包的时候,需要一个变量来接收,否者的话返回的函数会作为一个垃圾对象被回收(并没有任何变量引用它,所以会被回收),函数里面的变量也就不存在了

例如

fun1()();  //1
fun1()();  //1
fun1()();  //1

# 销毁

其实销毁很简单,只需要将变量设置为null就可以了

let fun = fun1();
fun();  //1
fun = null;

        fun: function (m) {
            return fun(m, n);
        }
    }
