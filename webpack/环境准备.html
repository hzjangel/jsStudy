# 环境准备

# nodejs

可以使用nvm安装nodejs,也可以下载安装包安装nodejs

安装好nodejs之后,可以设置淘宝镜像

npm config set registry https://registry.npm.taobao.org

不要把npm变为cnpm命令,在运行的工程中会出现一些问题

# webpack-cli

这个不要全局安装,直接本地安装就可以了

# 本地安装
npm install webpack --save-dev
# 全局安装(最好不要全局安装)
npm install webpack --global
